# MarchineLearning

# 1. Marchine Learning 개요
  * 역사, 문제점
  * 머신러닝 종류
      - 지도학습 : 데이터에 대한 Label(명시적인 답)이 주어진 상태에서 학습
      - 비지도학습 : 데이터데 대한 Label(명시적인 답)이 없는 상태에서 학습
      - 강화학습 : 지도학습과 비슷하지만 완전한 답(Label)을 제공하지 않음
